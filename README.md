# 📈 Finance-ClawBot (lh-wx-helper)
**个人金融风控与盯盘助手 (SRE 级高可用版)**

这是一个基于 SRE（站点可靠性工程）理念设计的轻量级个人金融监控工具。它旨在消除市场噪音，通过智能网格策略和精准的节假日过滤，在触发关键风控节点时，通过飞书（Lark）互动卡片向你发送高信噪比的报警与数据看板。

## ✨ 核心特性

* **📊 极简收盘日报**：过滤无关紧要的盘中波动，每日定点推送核心宽基、ETF 的收盘数据。
* **🚨 黄金智能风控**：采用“非对称网格策略”（涨幅 ≥ 1.0% 或跌幅 ≤ -1.0% 触发报警），完美过滤日常窄幅震荡。
* **🧠 断电记忆 (持久化)**：使用本地 `json` 状态机记录黄金日内基准价，即使容器重启也不会丢失涨跌幅计算锚点。
* **😴 节假日防骚扰**：内置中国法定节假日与调休逻辑（基于 A 股真实开盘规则），周末及休市期间自动静默。
* **🎨 飞书终端级卡片**：彻底抛弃纯文本，采用飞书高级互动卡片（Grid 布局），实现红涨绿跌、严丝合缝的金融级数据看板。

## 💡 常见问题：为什么计算出的“国内金价”和京东金融/支付宝不一样？

本工具推送的 **国内折合价 (AU9999/克)** 采用的是国际标准换算公式：`(国际现货金价 ÷ 31.1035 × 离岸人民币汇率)`。
如果你发现它与国内看盘软件显示的绝对价格有差异，主要由以下三大底层金融逻辑导致：

1.  **在岸 (CNY) vs 离岸 (CNH) 汇率差**：
    国内软件使用受央行指导的“在岸人民币(CNY)”，而本工具使用国际自由市场流通的“离岸人民币(CNH)”。两者存在正常的价差（几十个基点），在极端行情下会进一步拉大。
2.  **实时中间价 vs 银行挂牌价**：
    瑞讯银行提供的是国际外汇市场每秒跳动的**真实交易中间价**。而国内银行或平台显示的价格，通常包含了他们需要赚取的“点差”（买入/卖出差价）。
3.  **核心原因：国内黄金溢价 (Shanghai Premium)**：
    受限于严格的黄金进口配额管制，国内实物黄金长期处于“供不应求”的状态。因此，上海黄金交易所的 AU9999 现价通常比国际金价**贵 1~3 元/克**（极端时甚至超过 10 元）。国内软件显示的是包含溢价的交易价，而本工具计算的是**“无水分的纯理论折合价”**。

> **风控提示**：作为盯盘机器人，我们更关注的是**波动率（涨跌幅百分比）**而非绝对价格。使用理论折合价不仅能准确反映市场趋势，还能帮助你判断当前国内金价是否存在“高溢价泡沫”。
